My Projects

Below is a collection of various electrical engineering and tech projects I've worked on.

Home Lab Power Supply

A custom variable power supply built for my electronics home lab.

This project features:

  • Adjustable voltage output (0-30V)
  • Current limiting (0-3A)
  • Digital display for voltage and current
  • Overheat protection
  • Short circuit protection

LED Matrix Display

An 8x8 RGB LED matrix with Arduino-controlled patterns and animations.

Features include:

  • 64 individually addressable RGB LEDs
  • Multiple animation patterns
  • Sound reactivity mode
  • Custom 3D-printed enclosure
  • Web interface for pattern selection

IoT Weather Station

A wireless weather station with temperature, humidity, and pressure sensors reporting to a custom dashboard.

Features include:

  • Real-time temperature, humidity, and barometric pressure monitoring
  • Solar-powered with battery backup
  • WiFi connectivity for data transmission
  • Custom web dashboard for data visualization
  • Historical data tracking and graphing
